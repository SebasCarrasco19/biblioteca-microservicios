## book-service

Se incorporo el microservicio book-service a la arquitectura general del sistema.
Se implemento una estructura por capas (Controller, Service y Repository), persistencia con MySQL.
Se integro manejo centralizado de excepciones con respuestas HTTP consistentes.
Este avance fortalece la cohesion del dominio de biblioteca y cumple con los lineamientos tecnicos definidos para la entrega parcial.
